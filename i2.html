About Me

I design and build Minecraft Bedrock addons: data-driven entities, new items, UI systems and skin packs. I aim for compatibility, easy install, and behaviour that feels "native" to Bedrock. I publish community-friendly documentation and provide support via Discord and Patreon.

  • Tools: Blockbench, Visual Studio Code, JSON behaviours, Bedrock scripting API (learning)
  • Specialities: Lightweight systems, modular items, custom UIs
  • Available for contracts, marketplace partnerships, and commissions
